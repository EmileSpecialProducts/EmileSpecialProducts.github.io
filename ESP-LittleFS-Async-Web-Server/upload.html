ESP-LittleFS-Async-Web-Server

Devices ESP8266-12E ESP-32 (S3/S2/C3/C6)

This will program the ESP-LittleFS-Async-Web-Server software to the ESP8266-12E ESP-32 (S3/S2/C3/C6) device.
After programming you can connect to the AP Wifi name is ESP-xxxxxx and set the Wifi SSID and Password.
After reboot you can connect to the device. ( sometimes you will need a second reboot )
